Benjee10sharedAssets 0.1
A part library for Kerbal Space Program 1.8.0+ by Benjee10.

INSTALLATION
Place the contents of GameData/ into your KSP/GameData/ directory and install necessary dependencies.

DEPENDENCIES
This mod is used by Habtech2, SOCK and reDIRECT. Dependencies of these mods will be needed. This library should not be installed on its own.
